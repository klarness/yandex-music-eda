# Анализ поведения пользователей Яндекс Музыки
EDA на основе открытого датасета Yambda от Яндекса.

## Данные

Датасет опубликован Яндексом на Hugging Face:
[yandex/yambda](https://huggingface.co/datasets/yandex/yambda), subset `flat-multievent-50m`
Файлы датасета в репозиторий не включены, использовался `listens.parquet` по ссылке выше

47.8 млн событий прослушивания, 9 238 пользователей, 877 168 уникальных треков.



## Что исследовалось:

1. Дослушивают ли пользователи треки из рекомендаций чаще, чем найденные самостоятельно
2. Зависит ли дослушиваемость от длины трека
3. Есть ли небольшая группа пользователей, которая составляет большую часть всех прослушиваний

## Результаты

| Гипотеза | Результат |
|---|---|
| Органику дослушивают чаще рекомендаций | **Не подтвердилась** рекомендации дослушивают чаще (65% vs 54%) |
| Короткие треки дослушивают чаще длинных | **Подтвердилась**  дослушиваемость падает с 64% до 52% |
| Меньшинство генерирует большинство прослушиваний | **Подтвердилась**  16% пользователей дают 50% всех прослушиваний |
